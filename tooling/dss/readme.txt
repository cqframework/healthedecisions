Example Implementation

This folders contains a sample implementation of a RESTful web service using the HL7 DSS and vMR standards.

NOTE: This is informative content only.

The following projects are included as part of the implementation:

HeD.CDSS.Model - DSS IG models, generated from the kaoutput.xsd schema
HeD.DSS - ASP.NET RESTful web service
HeD.DSS.Models - DSS Standard models, generated from the DSS OMGDssSchema.xsd
HeD.DSS.SampleClient - Sample client consuming the RESTful web service
HeD.Model - HeD action models, generated from the HeD action.xsd schema
